= Jaffascript Roadmap

This roadmap tracks Jaffascript development. Remember: Jaffascript is a *brand surface only*.
Core language features are developed in the link:https://github.com/hyperpolymath/affinescript[affinescript] repository.

== Current Status

Alpha. The jaffascript face is established with:

* CLI shim ()
* justfile with common tasks
* Examples in 
* Documentation in  and 
* EXPLAINME.adoc with terminology mapping

== Milestones

=== Phase 1: Face Transformer (Upstream in affinescript)
* [ ] Implement  text transformer
* [ ] Add jaffascript variant to  face type
* [ ] Add jaffascript error vocabulary to  format functions
* [ ] Register jaffascript in  parse_with_face
* [ ] Add jaffascript to  name aliases
* [ ] Create  debug command

=== Phase 2: Documentation
* [x] README.adoc with overview and examples
* [x] EXPLAINME.adoc with terminology mapping
* [ ] Quickstart guide for jaffascript developers

* [x] Basic hello world example
* [ ] Data structures example (lists, records, variants)

=== Phase 4: Testing
* [ ] Face transformer regression tests
* [ ] Examples round-trip through transformer

=== Phase 5: Tooling (Optional)
* [ ] Language server integration
* [ ] Syntax highlighting

== Dependencies

All milestones depend on upstream affinescript development. Jaffascript cannot release
independently of affinescript.
